What is an output?

An output in JavaScript is a way to display information. Outputs can be used to display text, or information stored inside of variables. It can even display the result of a mathematical equation.

How to write an output:

There are a comple of different ways to use an output in JavaScript. One of the simplest ways is by using the line document.write(info goes here); In this line, you are telling the computer to write whatever is inside the parentheses. To try out using outputs, Click here!

What is a variable?

In JavaScript, variables are used to display and data that is stored inside of it. A variable is like a labeled box. You know what the contents are if you label the box correctly, but you can only see what is inside the box by opening it.

How to write a variable:

Writing a variable in JavaScript is extremely easy. There is no need to define what type of variable is inside, as the language automatically determines that. Simply use the line var and then follow it with the name. Be careful, as the name cannot contain numbers or symbols as the first character. To test out variables, Click here!

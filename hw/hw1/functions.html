What is a function?

A function is a piece of code that allows for a specific task to be completed. Functions are often useful when multiple steps need to be taken to get a certain result.

How to write a function:

First one must create the function by typing the word function, and then giving it a name. Then within parentheses, they can type different parameters such as numbers or strings. AFter that, curly brackets are placed and the action is written down.
